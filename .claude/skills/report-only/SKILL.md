# 🚨 보고 전용 모드 (Report Only Mode)

사용자가 대화 중에 **"보고해"**라는 키워드를 명시하거나 포함하여 지시할 경우, AI 에이전트는 다음 행동 수칙을 **모든 작업의 최우선 순위**로 두고 100% 준수해야 합니다.

## 1. 상태 변경 액션(Action) 절대 금지
- 파일을 수정(`replace_file_content`, `write_to_file` 등)하거나, 명령어를 자체적으로 실행(`run_command`)하거나, 브라우저를 조작(`browser_subagent`)하는 등 시스템 상태를 변화시키는 일체의 도구 호출을 즉시 금지합니다.

## 2. 순수 텍스트 보고만 수행
- 방금 전 사용자가 내린 지시사항이 무엇인지, 현재 에이전트가 어떤 상황과 문제점을 마주하고 있는지, 앞으로 어떻게 할 계획인지만을 분석하여 **오직 텍스트(마크다운)로만 답변**합니다.
- 어떠한 도구(Tool)도 병렬로 실행해서는 안 됩니다. (자료 조회를 위한 `view_file` 등 읽기 전용 도구는 예외)

## 3. 사용자 승인 대기 단계 체류
- 보고를 마친 후, 사용자가 명시적으로 "수정해", "진행해", "적용해" 등의 추가 액션 승인 명령을 내리기 전까지 스스로 판단하여 작업을 재개하거나 다음 스텝으로 넘어가지 않습니다.
