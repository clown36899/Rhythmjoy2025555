# 네이버 카페 강습·행사 수집 스킬 (Cafe Lesson Ingestion)

이 스킬은 스윙패밀리 네이버 카페에서 강습, 워크숍, 행사, 연습실 정보를 수집하여 `scraped_lessons.json`에 저장하는 표준 절차를 정의합니다.

## 📌 수집 대상 및 범위
- **소스**: 스윙패밀리 네이버 카페 [외강&행사&연습실] 게시판
- **URL**: [접속하기](https://cafe.naver.com/f-e/cafes/10342583/menus/13?viewType=L)
- **범위**: **첫 페이지에 보이는 모든 게시물** (15개)
- **유형**: 강습, 워크숍, 행사, 소셜, 연습실 — **모든 유형**

> [!CAUTION]
> **강제 수칙 (Enforcement Rules)**
> 1. **로그인 금지**: 로그인 없이 조회 불가 시 해당 게시물은 즉시 스킵합니다.
> 2. **첫 페이지만**: 2페이지 이동이나 페이지네이션 클릭을 절대 금지합니다.
> 3. **검색/필터 금지**: 게시판 내 검색이나 말머리 필터를 변경하지 않습니다.
> 4. **이미지 다운로드 우선**: CDN URL을 추출하여 `curl -o`로 직접 다운로드합니다. 실패 시에만 캡처합니다.
> 5. **ID 규칙**: `swf_YYMMDD_[유형약어]` 형식을 준수합니다. (예: `swf_260220_workshop`)

## 🛠️ 수집 공정

1. **목록 확보**: 첫 페이지 15개 게시물의 제목, 번호, 작성일 추출하여 `task.md`에 기록.
2. **개별 게시물 처리**:
   - **1 게시물 = 1 데이터**: 가장 빠른 시작일 기준 1개 데이터만 생성 (날짜별 쪼개기 금지).
   - **이미지**: `public/scraped/`에 저장 (CDN 우선).
   - **텍스트 파싱**: 제목, 유형, 시작일(YYYY-MM-DD), 장소, 강사/DJ, 비용, 신청 링크 추출.
3. **데이터 기록**: `src/data/scraped_lessons.json` 업데이트.
4. **동기화 및 확인**:
   - 이미지 파일 무결성 확인.
   - [관리자 인제스터](http://localhost:8888/admin/ingestor)의 **🎓 강습/워크숍** 탭에서 최종 확인.

## 작업 개시 시 task.md 템플릿

```markdown
# 강습/행사 수집 (스윙패밀리 카페)

- [ ] 카페 접속 및 목록 확보 (첫 페이지 15개)
- [ ] 상세 수집 및 파싱 (게시물별 체크)
    - [ ] 게시물 1: [제목]
    - [ ] 게시물 2: [제목]
    - ...
- [ ] 데이터 동기화 (`scraped_lessons.json`)
- [ ] 관리자 페이지 최종 확인
```
