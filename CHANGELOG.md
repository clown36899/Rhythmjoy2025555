# 변경 이력 (버전별)

## [3.1.2] - 2025-12-31
### 🚀 성능 최적화
- **API 호출 구조 획기적 개선**
  - 초기 로딩 시 누수되던 `app_settings` 테이블 조회를 통합 RPC(`get_board_static_data`)로 일원화
  - 사용자의 상호작용(좋아요, 즐겨찾기 등) 데이터를 전역 Context에서 공유하여 페이지 이동 시 중복 호출 제거
  - 이벤트 목록 및 소셜 스케줄에 SWR(Stale-While-Revalidate) 캐싱 도입으로 실시간성 유지 및 로딩 속도 향상
  - 게시판 권한 체크 로직 통합으로 불필요한 RPC 호출 제거

### 🔧 개선 사항
- `BoardDataContext` 확장으로 전역 데이터 관리 효율성 증대
- `useSocialSchedulesNew`, `useUserInteractions` 훅 리팩토링 및 캐싱 레이어 추가

## [3.1.1] - 2025-12-30
### 🐛 버그 수정
- **연습실 카드 텍스트 겹침 문제 해결**
  - 카드 내부 패딩 및 간격 최적화 (1.5rem → 0.875rem)
  - 텍스트 크기 및 줄 높이 조정으로 고정 높이 내 안정적 표시
  - 모든 텍스트 요소에 overflow 처리 적용

## [3.1.0] - 2025-12-30
### ✨ 새로운 기능
- **이벤트 이미지 고유 폴더 관리 시스템 도입**
  - `event-posters/{timestamp}_{random}/` 구조의 격리된 폴더 시스템 적용
  - 업로드된 모든 사이즈(micro/thumb/med/full)를 한 폴더에서 관리
- **storage_path 데이터베이스 연동**
  - Supabase Storage의 폴더 경로를 DB에서 추적하여 관리 효율성 증대
- **이전 이미지 자동 클린업 기능**
  - 이미지 수정 시 이전 폴더 및 레거시 개별 파일 자동 삭제 로직 추가
- **이벤트 삭제 로직 개선**
  - 이벤트 삭제 시 스토리지의 관련 폴더 전체를 동시에 삭제하도록 보완
