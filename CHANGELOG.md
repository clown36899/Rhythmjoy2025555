# 변경 이력 (버전별)

## [2.23.0] - 2025-12-29
### ✨ 새로운 기능
- **소셜 페이지 금주의 일정 개선**
  - 오늘 일정이 없을 때 가장 가까운 날짜로 자동 선택
  - "오늘" 텍스트를 노란색으로 강조 표시
  - 전체일정 탭에서 과거 일정 자동 필터링 (오늘 포함 미래만 표시)
- **행사 다중 일정 지원**
  - 여러 날짜에 걸친 행사가 금주의 일정에서 각 요일별로 표시
- **하단 메뉴 개선**
  - "소셜" 탭 이름을 "소셜 /동호회"로 변경
  - 메뉴 크기 조정 및 반응형 디자인 개선
- **V2 페이지 동호회 강습 섹션**
  - "동호회 등록" 버튼 추가로 소셜 페이지 바로 이동 가능

### 🐛 버그 수정
- 자유게시판 말머리(prefix) 조회 오류 수정

## [2.22.0] - 2025-12-28
### ✨ 새로운 기능
- **EventDetailModal 인라인 날짜 편집 기능 추가**
  - 이벤트 상세 보기에서 연필 아이콘 클릭 시 날짜를 직접 수정 가능
  - 하루/개별 날짜 선택 모드 지원 (강습/동호회 제한)
  - DatePicker UI 제공
  - 날짜 변경 시 저장 확인 프롬프트 표시

### 🐛 버그 수정
- **날짜 시간대 오프셋 문제 해결**
  - DatePicker에서 UTC 변환으로 인한 날짜 하루 어긋남 현상 수정
  - 로컬 날짜 포맷팅으로 선택한 날짜 정확히 저장
- **날짜 편집 상태 관리 개선**
  - `dateMode` state 추가로 날짜 모드와 값 분리
  - `editValue` 충돌 해결
- **변경 감지 함수 업데이트**
  - `hasChanges()` 함수에 날짜 필드 추가
- **DatePicker 드래그 선택 방지**
  - 단일 날짜 모드에서 드래그로 범위 선택되던 문제 수정
- **EventRegistrationModal 날짜 저장 문제 해결**
  - 이벤트 수정 시 날짜가 저장되지 않던 문제 해결

### 🔧 개선 사항
- **종합 디버깅 로깅 시스템 추가**
  - 날짜 선택, 저장, DB 업데이트 전 과정 추적
- **날짜 필드 검증 로직 강화**
  - 저장 성공 여부 확인 강화
